PluginContext Events

The events of the PluginContext class are listed below. For a complete list of PluginContext class members, see the PluginContext Members topic.

Public Instance Events

AfterPluginsStarted Fires after all of the Plugins have been started
BeforePluginsStopped Fires before all of the Plugins are stopped
PluginStarted Fires when a Plugin is started
PluginStopped Fires when a Plugin is stopped
