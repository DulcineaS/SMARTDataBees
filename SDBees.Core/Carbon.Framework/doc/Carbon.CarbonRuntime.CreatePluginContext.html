CarbonRuntime.CreatePluginContext Method 

Starts and initializes the Carbon Runtime with a new PluginContext. The new PluginContext will be executed in the current AppDomain, which will create and run a new ApplicationContext. This is the Carbon entry point that should be called by the hosting executable.
