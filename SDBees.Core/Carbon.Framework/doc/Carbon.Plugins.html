Class Description
Plugin Defines the base class from which all Plugin classes must derive.
PluginApplicationContext Provides the contextual information about an application thread.
PluginContext Defines an application context in which Plugins are hosted.
PluginContext.PluginContextAlreadyExistsException Defines an exception that is generated as a result of attempting to create more than one PluginContext per application.
PluginContext.PluginContextAlreadyRunningException Defines an exception that is throw by the PluginContext class if the Run method is called more than one time.
PluginContext.PluginContextException Defines the base PluginContext exception that is thrown by the PluginContext class.
PluginDescriptor Provides a class that describes a single Plugin instance using meta-data and contains the active instance of the Plugin type.
PluginDescriptorCollection Provides a strongly-typed collection of PluginDescriptor instances. This class is thread safe.
PluginDescriptorCollection.PluginDescriptorAlreadyExistsException Defines an Exception that is generated by the PluginDescriptorCollection when a PluginDescriptor is added to the collection for a Type when another PluginDescriptor already exists in the collection for the same Type.
PluginDescriptorEventArgs Defines an EventArgs class for the PluginDescriptorEventHandler delegate.
